h2. หมายเหตุ

* กรณีที่ Build Feature ทดสอบภายใน สามารถกำหนด Branch เองได้ แต่ Tag Build/ARTIFACT จะถูก Clear ทุกๆ 3 เดือน
* กรณีที่ Build for Internal / QA ใช้ Branch [BNZTARGETVERSION]X.YY.Z 
* กรณีที่ Build for Customer ใช้ Master 
* waitfor9Com-INVEST-SERVICE เป็น Docker Image ที่จะเอามา Replace WinServ
* waitfor9Com-APP-SERVICE Build ตอนที่เป็น Customer Version หรือ มี Defect ที่้ต้องส่ง QA
